# GitHub User Finder

**Автор:** Мукатаева Самида

## Описание

GitHub User Finder — это графическое приложение для поиска пользователей GitHub через официальное API. Позволяет просматривать информацию о пользователе (имя, количество репозиториев, подписчиков и т.д.), добавлять пользователей в избранное и управлять списком избранных.

## Как использовать API

Приложение использует публичное GitHub REST API:
- Endpoint: `https://api.github.com/users/{username}`
- Метод: GET
- Ответ: JSON с информацией о пользователе

API не требует авторизации для поиска публичных пользователей.
